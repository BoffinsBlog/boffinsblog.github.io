Steps#

Ensure that the Debian Bookworm system is fully updated.

I ran through this process immediately after installing Debian Bookworm so I didn’t expect there would be much to upgrade but did it anyway.

As this upgrade was done on top of a fresh installation there was nothing for me to backup, nor were there additional PPA’s and repositories to worry about updating.

YMMV following my steps.

Note: During the upgrade processes there are in-Terminal messages, keep an eye on the screen.

Become a superuser:

su

Perform a full upgrade:

apt update

apt upgrade

apt full-upgrade

Clean any leftover packages and reboot:

apt --purge autoremove

reboot

Update the Bookworm repositories to Trixie:

sed -i 's/bookworm/trixie/g' /etc/apt/sources.list

Update the repository use the newly added repositories:

apt update

The Debian documentation says the recommended way to handle the upgrades is in two (2) separate steps:

  • Minimal system upgrade

  • Full system upgrade

First, upgrade all currently installed packages to their latest versions but without installing any new packages:

apt upgrade --without-new-pkgs

Then, once the minimal system upgrade completes, perform a full upgrade:

apt full-upgrade

There were quite a few warning messages:

dpkg: warning: unable to delete old directory

Nothing catastrophic that prevents the ‘Plug from booting and running.

Conclusion#

It may be prudent to wait until there are Trixie installer images for the ‘Plug but in the meantime this seems to be running OK.
