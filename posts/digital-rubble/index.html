Digital Rubble

October 8, 2025 · 2 min · Boffin

So, I used to have a website.

A proper domain, mail server, everything.

Then, one day I decided the upkeep was all a bit too much .

So I binned it and decided to host my humble pages on my ISP provided fixed IP.

It kinda worked but not very well.

(Some things just won’t work without a domain.)

So now I have a new plan.

I bought a cheap £5 (for the first year) domain.

Grabbed a matching email.

Opened a matching GitHub account.

Now, when the domain comes up for renewal and it’s silly money, I can relocate the domain content to GitHub Pages .

(I’d already moved from WordPress to Hugo some time ago and apparently Hugo can be hosted on GitHub Pages quite easily.)

I realised that I had been getting hung up on naming; losing a memorable domain name.

(I still kinda regret binning that last one; it was a good name.)

And then I thought, what if I just leave the GitHub Pages there?

Untouched, abandoned?

Maybe even delete the GitHub keys and associated email account.

I would never be able to update that bit of the Internet again.

It would be left, like a forgotten building in Microsoft’s city.

I could start afresh, get another £5 (for the first year) domain.

Create the email and GitHub accounts.

Rinse and repeat.

I would become a blogging vandal.

Leaving rubble on other peoples infra.
